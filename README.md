# Spotify Profile

Aplicação com diversas funcionalidades, principalmente focadas em exibir dados recebidos da API do Spotify.

## Funcionalidades

- **Profile Tab**: Visualizar as informações gerais como: nome, foto e seguidores.

## Tecnologias utilizadas

- React / Next
- Tailwind
- Typescript
- Spotify API
